### Prize Wheel

#### Visible

Determines if the object is rendered.

`Toggle switch`
`Default: ON`

#### Direction

Determines what direction the front of the ladder faces.

`Range input (float)`
`Range: 0-7`
`Default: 0`

#### Spin Type

`Dropdown`
`Default: Starter`
```
Options:
Starter
Elite
Heroic
```

#### Style > Encoding

`Dropdown`
`Default: Default`
```
Options:
Default
sRGBEncoding
LinearEncoding
```

#### Style > Shading

Enables shading. Disable if a flat look is preferred.

`Toggle switch`
`Default: ON`

#### Position

Location of the object in Cartesian coordinates. Positive Y is in the up direction. X and Z location defines the *center* and the Y location defines the bottom.

`Number input (float)`
`Default: 0, 0, 0`

#### Size

Scale of the object.

`Number input (float)`
`Default: 0, 0, 0`
